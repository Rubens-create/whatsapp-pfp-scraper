# WhatsApp Profile Picture Fetcher

Serviço para extrair a URL da foto de perfil do WhatsApp via Puppeteer.

## Funcionalidades
- Recebe um número de telefone via query parameter.
- Retorna a URL da foto de perfil em formato JSON.
